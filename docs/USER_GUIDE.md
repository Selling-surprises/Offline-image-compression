# 离线图片压缩工具 - 使用指南

## 🎯 功能特点

### ✨ 核心优势
- **完全离线处理**：所有图片压缩在浏览器本地完成，保护您的隐私
- **无需上传**：图片不会发送到任何服务器
- **PWA支持**：可安装为桌面或移动应用，支持离线使用
- **批量处理**：支持同时处理多张图片

### 📸 支持的格式
- JPEG
- PNG
- WebP
- AVIF

## 🚀 快速开始

### 1. 上传图片
- **拖放上传**：直接将图片拖放到上传区域
- **点击选择**：点击上传区域选择文件
- **批量上传**：可同时选择多张图片

### 2. 配置压缩设置

#### 输出格式
- **自动选择**（推荐）：根据图片特征自动选择最佳格式
- **JPEG**：适合照片和复杂图像
- **PNG**：适合需要透明背景的图片
- **WebP**：现代格式，体积小质量高
- **AVIF**：最新格式，压缩率最高

#### 压缩模式
- **高质量压缩**：保持原始分辨率，适度降低质量
- **感知无损压缩**：使用高级算法，视觉效果更好

#### 优化级别
- **轻度优化**：压缩率约 10-20%，质量损失最小
- **中度优化**：压缩率约 20-40%，质量与体积平衡
- **深度优化**：压缩率约 40-60%，体积最小

#### 分辨率设置
- **保持原始分辨率**：不改变图片尺寸
- **自定义尺寸**：设置最大宽度和高度限制

### 3. 开始压缩
点击"开始压缩"按钮，工具会自动处理所有待处理的图片。

### 4. 下载结果
- **单张下载**：点击每张图片卡片上的"下载"按钮
- **批量下载**：点击"下载全部"按钮下载所有已压缩的图片

## 📊 统计信息

工具会实时显示以下统计数据：
- **总图片数**：已上传的图片总数
- **已完成**：已成功压缩的图片数量
- **待处理**：等待压缩的图片数量
- **节省空间**：总共节省的文件大小
- **总压缩率**：整体压缩百分比

## 💡 使用技巧

### 最佳实践
1. **选择合适的格式**：
   - 照片使用 JPEG 或 WebP
   - 图标、Logo 使用 PNG 或 WebP
   - 追求最小体积使用 AVIF

2. **根据用途选择优化级别**：
   - 网页展示：中度或深度优化
   - 打印输出：轻度优化
   - 社交媒体：中度优化

3. **批量处理**：
   - 一次上传多张图片可以节省时间
   - 相同类型的图片使用相同设置

### 性能提示
- 大文件（>5MB）会自动使用 Web Worker 处理，不会阻塞界面
- 处理超时时间为 30 秒
- 建议单个文件不超过 50MB

## ❓ 常见问题

### Q: 为什么有些格式不可用？
A: 某些浏览器可能不支持 WebP 或 AVIF 格式。工具会自动检测并只显示支持的格式。

### Q: 压缩后的图片在哪里？
A: 压缩后的图片需要手动下载到本地。点击"下载"按钮即可保存。

### Q: 可以处理多大的文件？
A: 建议单个文件不超过 50MB。超过 5MB 的文件会自动使用 Web Worker 处理。

### Q: 压缩会损失质量吗？
A: 是的，压缩会有一定的质量损失。但通过合理选择压缩级别，可以在质量和体积之间找到最佳平衡。

### Q: 为什么需要 HTTPS？
A: PWA 功能（如 Service Worker）需要在 HTTPS 环境下运行，以确保安全性。
